# Gold Miner Clicker: MMORPG Edition

A premium, interactive Idle MMORPG game featuring player leveling progression, active hotbar skills (Q, W, E), hero worker guilds, and a cracking core vein prestige engine.

## Features

1. **Dynamic Core Vein Cracking**: The central click target is a multi-layered health pool vein (Copper -> Silver -> Gold -> Sapphire -> Titan Core). As you damage it, cracks appear and it shakes, ultimately fracturing with screen flashes, swept audio synthesizers, and massive rewards.
2. **Guild Hero Classes**:
   - **Dwarf Demolisher**: Active tap power booster. Unlocks **Demolish (Q)** (double clicks).
   - **Goblin Engineer**: Passive GPS booster with worker cost discount. Unlocks **Overclock (W)** (triple GPS speed).
   - **Crystal Sage**: Critical strike booster and dungeon speed booster. Unlocks **Focus (E)** (100% crit chance).
3. **Synergy Combos**: Trigger all three active skills at once to activate the **Ultimate Mining (Абсолютный бур)** combo for a flat +500% damage multiplier.
4. **Quest & Raid Log**: Scrolling live combat feed color-coded for critical taps, loot drops, and milestones.
5. **Mobile-First Multitouch Support**: Touch layout supporting multiple fingers tapping simultaneously with individual pickaxe visual animations.
6. **Native Android Package (APK)**: Pre-compiled and ready to install on Android devices.
