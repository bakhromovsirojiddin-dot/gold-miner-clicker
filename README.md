# Gold Miner Clicker & Android App

A premium, interactive Idle Clicker game featuring player leveling progression, active multitouch, miner expeditions, and a prestige system.

## Features

1. **Player Levels & Progression**: Earn XP from mining. Leveling up grants a permanent global +10% multiplier and unique Dwarven titles.
2. **Prestige Reset (Altar)**: Reach Level 20+ to trigger deep drilling. Resets active progress for **Ancient Diamonds** to purchase permanent blessings.
3. **Journal Achievements**: Complete milestones (clicks, auto-miners hired, boosts) for gold and diamond rewards.
4. **Miner Expeditions (Offline)**: Send miners on background expeditions for resources and artifacts (Provisions, Lucky Clover, Ancestral Pickaxe).
5. **Mobile Multitouch**: Fully optimized touch layout supporting multiple fingers tapping simultaneously with individual pickaxe visual animations.
6. **Native Android Package (APK)**: Wrapped using WebView asset loaders with DOM storage persistence.
